Here is a list of essential Linux commands with a direct example for each.
## 1. Navigation & Location

* pwd (Print Working Directory): Shows exactly where you are in the filesystem.
* Example: pwd (Outputs: /home/user/Documents).
* ls (List): Lists the files and folders in your current directory.
* Example: ls -l (Lists files in a detailed "long" format).
* cd (Change Directory): Moves you to a different folder.
* Example: cd /var/log (Moves you into the log folder). [1, 2, 3, 4, 5, 6] 

## 2. File & Directory Management [4] 

* mkdir (Make Directory): Creates a new folder.
* Example: mkdir new_folder.
* touch: Creates a new, empty file.
* Example: touch report.txt.
* cp (Copy): Copies a file or folder.
* Example: cp file.txt backup.txt (Creates a copy named backup.txt).
* mv (Move/Rename): Moves a file or changes its name.
* Example: mv oldname.txt newname.txt (Renames the file).
* rm (Remove): Deletes a file.
* Example: rm file.txt.
* rmdir: Deletes an empty folder.
* Example: rmdir old_folder. [2, 3, 4, 5, 7, 8, 9, 10, 11] 

## 3. Viewing & Searching Content

* cat (Concatenate): Displays the entire content of a file on your screen.
* Example: cat notes.txt.
* head: Shows the first few lines of a file.
* Example: head -n 5 file.txt (Shows the first 5 lines).
* tail: Shows the last few lines of a file.
* Example: tail -n 10 logs.txt (Shows the last 10 lines).
* grep: Searches for a specific word or pattern inside a file.
* Example: grep "error" syslog.txt (Finds every line with the word "error").
* find: Searches for files and directories based on their name.
* Example: find . -name "*.jpg" (Finds all JPEG files in the current folder). [2, 3, 4, 8, 12, 13, 14, 15, 16, 17] 

## 4. System & Permissions

* sudo (SuperUser Do): Runs a command with administrative (root) privileges.
* Example: sudo apt update.
* chmod (Change Mode): Changes the permissions of a file (who can read, write, or run it).
* Example: chmod 755 script.sh.
* top: Displays a real-time list of running processes and system resources.
* Example: top.
* df: Shows how much disk space is being used.
* Example: df -h (Shows space in "human-readable" MB/GB). [2, 4, 8, 11, 18]
